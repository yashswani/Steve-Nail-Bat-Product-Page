# Steve — Nail & Bat Product Page

A small static product landing page showcasing the "Steve" Nail & Bat product. This repository contains a minimal HTML/CSS demo intended for quick previews and lightweight customization.

## Preview
Open `index.html` in your browser to view the page locally.

## Built With
- Plain HTML (`index.html`)
- CSS (`styles.css`)

## Files
- `index.html` — main page markup
- `styles.css` — styles for the page

## Usage
1. Clone the repo or download the files.
2. Open `index.html` in any modern browser.

Optional: run a simple local server for proper file routing (useful for some browser setups):

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

## Development
- Edit `index.html` and `styles.css` to change content and styles.
- Consider using a live-reload extension or a dev server for faster iteration.
